CN:

JavaFX Scene Builder是一种可视化布局工具，它使用户可以快速设计JavaFX应用程序用户界面，而无需编写代码。用户可以将UI组件拖放到工作区，修改其属性，应用样式表，并且将在后台自动生成他们正在创建的布局的FXML代码。结果是一个FXML文件，然后可以通过将UI绑定到应用程序的逻辑来与Java项目组合。

EN:

JavaFX scene builder is a visual layout tool, which enables users to quickly design the user interface of JavaFX applications without writing code. Users can drag and drop UI components to the workspace, modify their properties, apply style sheets, and automatically generate the fxml code of the layout they are creating in the background. The result is an fxml file that can then be combined with a java project by binding the UI to the logic of the application.

install[安装]

[en]After the normal software installation process, open eclipse and select Window > references > JavaFX > Browse... - > [path of JavaFX scene builder.exe you installed] [example: D:: (JavaFX) Java scene Builder 2.0 / JavaFX scene builder 2.0.exe) - > OK

[cn]前往 https://www.oracle.com/java/technologies/javafxscenebuilder-1x-archive-downloads.html下载最新版的JavaFX Scene Builder的安装包

通过正常的安装软件的流程安装完成后,打开Eclipse,选择Window -> References -> JavaFX -> Browse... -> [你安装的JavaFX Scene Builder.exe的路径]【例子：D:\javafx\Java Scene Builder 2.0\JavaFX Scene Builder 2.0.exe】 -> ok
